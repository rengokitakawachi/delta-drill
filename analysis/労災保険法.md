# 労働者災害補償保険法 分析ノート

## サマリー
労災保険法は「目的→災害認定→給付基礎日額→各種保険給付→通則→社会復帰促進等事業→特別加入→不服申立て・時効」という流れで構成される。択一・選択ともに得点源は **各種保険給付の数値(日数分・割合・限度額)** と **業務上/通勤の認定基準**、そして **特別加入の範囲**。本テキスト本文に厳密に基づき、選択式カード83枚を作成した(全17章をカバー)。数字の取り違えを誘う問題が多いため、「給付基礎日額の○日分」「100分の○」「○年○か月」を横断的に固める学習が有効。

## 頻度ランキング(過去問・出題比重ベース)
1. 業務災害・通勤災害(認定基準) — 災害認定章が過去問でも最大ボリューム
2. 通勤災害(通勤の定義・逸脱中断・複数就業者) — 出題常連
3. 障害(補償)給付(等級・併合繰上げ・前払/差額一時金)
4. 遺族(補償)給付(受給資格・順位・転給・年金額)
5. 休業(補償)給付(60%・待期3日・部分算定日)
6. 特別加入(3類型の範囲・要件・適用)
7. 給付基礎日額(平均賃金・スライド・年齢階層別限度額)
8. 通則・その他(費用徴収・第三者行為・支給制限・二次健診)
9. 特別支給金(休業20%・各支給額・ボーナス特別支給金)
10. 傷病(補償)年金(1年6か月・傷病等級・職権支給)

## 重要度マップ(weights連動)
- 重要度3(最頻出): 業務災害・通勤災害 / 休業(補償)給付 / 障害(補償)給付 / 遺族(補償)給付 / 通勤災害 / 特別加入
- 重要度2(頻出): 給付基礎日額 / 療養(補償)給付 / 傷病(補償)年金 / 介護(補償)給付 / 複数業務要因災害 / 通則・その他 / 特別支給金 / 不服申立て・その他
- 重要度1(基礎固め): 総則 / 葬祭料 / 社会復帰促進等事業

## 頻出論点 要点
### 目的(法1条)
- 「迅速かつ公正な保護」「労働者の福祉の増進に寄与」が選択式キーワード。政府管掌、被保険者概念なし。

### 業務上の認定
- 業務遂行性(事業主の支配下)+ 業務起因性(相当因果関係)。起因性の前提に遂行性。
- 脳・心臓疾患: 長期間=発症前おおむね6か月、時間外おおむね45時間で関連性が強まり80/100時間で「強」。
- 精神障害: 発病前おおむね6か月、ストレス‐脆弱性理論。

### 通勤の定義(法7条2項)
- 「就業に関し」「合理的な経路及び方法」、業務の性質を有するものを除く。
- 移動3類型: 住居↔就業の場所/事業場間移動/単身赴任の住居間移動。
- 逸脱・中断: 原則その後も通勤としないが、日常生活上必要な行為(則8条)をやむを得ない事由で最小限度に行う場合は、その間を除き通勤。ささいな行為は逸脱中断とされない。

### 給付基礎日額
- 原則=労働基準法12条の平均賃金。端数は1円に切上げ。
- 休業給付基礎日額: 10%超変動でスライド、療養開始1年6か月経過後に年齢階層別限度額適用。
- 年金給付基礎日額: 完全自動賃金スライド、支給開始当初から年齢階層別限度額。特別加入者は年齢階層別限度額不適用。

### 主要給付の数値(暗記必須)
- 休業: 給付基礎日額の60%、待期3日、休業特別支給金20%。
- 傷病年金: 療養開始後1年6か月、職権支給、第1級313/第2級277/第3級245日分。
- 障害: 1〜7級=年金、8〜14級=一時金。併合繰上げ(13級以上で1級・8級以上で2級・5級以上で3級)。
- 遺族年金: 1人153(妻単身55歳以上等175)/2人201/3人223/4人以上245日分。転給あり。
- 遺族一時金・前払上限: 給付基礎日額の1000日分。
- 葬祭料: 315,000円+30日分 と 60日分 の高い方。

### 特別加入
- 中小事業主(金融保険不動産小売50人/卸売サービス100人/その他300人)+事務組合委託、一人親方は団体加入、海外派遣は事務組合不要。
- 特別加入者: 二次健診なし、ボーナス特別支給金なし、年齢階層別限度額なし。

### 不服申立て・時効
- 審査官(審査請求・3か月以内)→ 労働保険審査会(再審査請求・2か月以内)。審査請求前置。
- 時効: 療養/休業/葬祭/介護/二次健診=2年、障害/遺族=5年。時効は基本権。

## 近年傾向
- 複数業務要因災害(令2.9.1施行)関連が定着。給付名称(複数事業労働者○○給付)と業務災害との対比が問われやすい。
- 認定基準は令和5年改正(脳・心臓 令5.10.18基発1018第1号 / 精神障害 令5.9.1基発0901第2号)に準拠。時間外労働の数値基準は要注意。
- 特別加入の対象拡大(特定フリーランス、情報処理・アニメ制作・芸能関係等)が新傾向。

## 学習アクション
1. 数字カードを横断暗記: 「給付基礎日額の○日分」「100分の○」「○年○か月」を一覧化して反復。
2. 業務上/通勤の認定は具体例(事例・通達)で○×判断を訓練。逸脱中断の例外5項目と「ささいな行為」を区別。
3. 給付ごとに「業務災害=補償あり/複数業務要因/通勤=補償なし」の名称対応表を自作。
4. 特別加入は3類型の要件(事務組合の要否・包括加入・団体加入)を表で整理し、通勤災害不適用者を覚える。
5. 時効2年/5年と不服申立ての期間(3か月/2か月)はセットで暗記。
6. weights重要度3の章(認定・休業・障害・遺族・通勤・特別加入)を優先周回。

# 🏦 厚生年金保険法 ― 出題傾向と重みづけ

> [[README]] に戻る

## 1. サマリー（結論）
- **核は「3つの年金」＝老齢厚生年金・遺族厚生年金・障害厚生年金**。中でも **老齢厚生年金が最頻出**。
- **被保険者・適用事業所**（適用拡大／70歳以上／高齢任意加入）も毎年級の頻出。
- **総合問題・選択式**は老齢・遺族・在職を横断して問うため、これらの論点の理解が得点に直結。
- 近年（R3〜R6）は**1年あたりの収録が急増**＝直近重視。**R5は事例当てはめ型の新傾向**（各給付の支給要件を横断的に思い出す力）。

## 2. 年度別 出題数（教材収録）
| 年度 | 出題数 |
|---|---|
| H19〜H30 | 各1問前後 |
| R1・R2 | 各5問 |
| **R3・R4・R5・R6** | **各10問**（直近重視） |

→ **直近5年（R1〜R6）で全体の約7割**。古い年度より**近年の傾向を優先**すべき。

## 5. 🔥頻出論点の"押さえどころ"
### 老齢厚生年金（★3）
- 支給要件：**65歳以上＋被保険者期間1か月以上＋老齢基礎の受給資格期間（10年）**
- **加給年金額**：被保険者期間**20年以上**＋生計維持の配偶者（65歳未満）・子。配偶者加給に**特別加算**
- **在職老齢年金**：基本月額＋総報酬月額相当額が**支給停止調整額（改正で変動）**超で一部停止
- **繰上げ（1月0.4%減）・繰下げ（1月0.7%増、最大75歳）**

### 適用事業所・被保険者（★3）
- **適用拡大**（短時間労働者の要件：週20時間・月8.8万円・2か月超・学生除く・規模要件）
- **70歳以上**は原則被保険者でない（高齢任意加入で例外）／**任意単独被保険者**は事業主同意要

### 遺族厚生年金（★3）
- 受給権者の**順位と範囲**（配偶者・子・父母・孫・祖父母／**夫は55歳以上**）
- **中高齢寡婦加算**（40歳〜65歳、要件）／**短期要件・長期要件**で計算が違う

## 6. 近年の注意（傾向の変化）
- **R5：事例当てはめ型**。障害・遺族の各年金の支給要件を横断で想起させる出題 → 単純暗記でなく**体系整理**が必要。
- 改正論点（**在職老齢年金の基準額・適用拡大・育休/産前産後の保険料免除**）は要注意。

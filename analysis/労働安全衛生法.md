# 労働安全衛生法 過去問分析ノート

## サマリー
労働安全衛生法は択一式で3問のみ出題され、多くの受験生が3問中1問程度しか正解できない難関科目。情報量が多く身近でないため、基本的な考え方をつかんだうえで「出題頻度の高い項目を集中的に学習し、用語や要件としての数値を最終的にしっかり覚える」のが最も効果的(基礎テキストp.12)。過去問講座の分布を見ると、出題は「安全衛生管理体制」と「健康の保持増進(健診・面接指導・ストレスチェック)」の2分野に集中している。選択式では、各種管理者・委員会の選任/設置の規模・人数、巡視頻度、健診・面接指導・ストレスチェックの頻度や日数、計画届の届出期限といった数値が狙われやすい。

## 頻度ランキング(過去問講座 章別収録数)
1. 安全衛生管理体制(3章) … 7問 ★最頻出
2. 健康の保持増進=健康診断・健康管理(7章) … 6問
3. 総合問題(横断) … 5問
4. 労働者の危険・健康障害防止(4章) … 4問
5. 機械等・危険物・有害物の規制(5章) … 3問
6. 総則・目的(1章) … 2問
6. 就業制限・安全衛生教育(6章) … 2問
6. 監督等=その他(9章) … 2問
9. 労働災害防止計画(2章) … 過去10年該当なし
9. 改善措置等(8章) … 収録なし

## 重要度マップ(章リスト基準)
- ★★★ 安全衛生管理体制 … 毎年のように出題。選任規模・人数・専属/専任・巡視頻度の数値が核
- ★★★ 健康診断・健康管理 … 健診の種類/頻度、面接指導(80h/100h)、ストレスチェック(50人)が定番
- ★★ 総則・目的 … 目的条文・定義・事業者概念。選択式で目的の穴埋めが王道
- ★★ 機械等・危険物・有害物の規制 … 検査証/型式検定の有効期間、製造禁止/許可の許可権者
- ★★ 就業制限・安全衛生教育 … 特別教育・職長教育の対象、就業制限、記録保存3年
- ★★ その他(計画届・監督・罰則) … 計画届の届出期限(30日/14日)、罰則の重さ
- ★ 労働者の危険・健康障害の防止 … 措置義務の総則的内容。横断・総合問題で問われる

## 頻出論点の要点(数値はすべて頻出)
### 選任/設置の規模(人)
- 総括安全衛生管理者:屋外的業種100／中間業種300／その他1,000(法10条、令2条)
- 安全管理者:対象業種50(法11条、令3条)/専任は建設業等300
- 衛生管理者:全業種50(法12条、令4条)/専任は1,000超 or 500超＋有害30
- 産業医:全業種50(法13条、令5条)/専属は1,000以上/2人は3,000超
- 安全委員会:業種により50 or 100(法17条、令8条)
- 衛生委員会:全業種50(法18条、令9条)
- 統括安全衛生責任者:常時50(危険性高い仕事は30)(法15条、令7条)

### 巡視頻度(混同注意)
- 安全管理者:規定なし／衛生管理者:毎週1回／産業医:毎月1回(一定の場合2か月に1回)/店社安全衛生管理者:毎月1回／特定元方:毎作業日(p.65)

### 健康診断・面接指導・ストレスチェック
- 定期健診:1年以内ごと/特定業務従事者・特殊健診:6か月以内ごと
- 健診個人票の保存:原則5年(石綿40年・特化物ベンゼン等30年)
- 健診結果報告:50人以上で実施。特殊健診・歯科健診は人数不問
- 面接指導:長時間労働者は80時間超＋疲労蓄積＋申出/研究開発・高プロは100時間超・申出不要
- ストレスチェック:1年以内ごと・50人以上で報告。50人未満は当分の間努力義務。記録は5年

### 計画の届出期限
- 機械等設置等:30日前(署長)/特に大規模建設:30日前(厚労大臣)/一般の建設・土石採取:14日前(署長)(法88条)

### 罰則の軽重
- 最重:製造等の禁止違反(3年以下拘禁刑 or 300万円以下)
- 選任義務・委員会設置義務違反:50万円以下の罰金/両罰規定あり(法122条)

## 近年傾向(R1〜R6)
- 「安全衛生管理体制」「健康保持増進」からの出題が継続して厚い(R3・R4・R5・R6で複数)。
- R6は8章相当(管理体制)・9章(監督等)から出題。管理者の選任要件・数値の正誤判定が中心。
- 総合問題(10章)も毎年級に複数収録され、章をまたぐ横断知識が問われる傾向。
- 化学物質の自律的管理(化学物質管理者・保護具着用管理責任者・リスクアセスメント)は近年の改正論点で要注意。

## 学習アクション
1. まず「選任/設置の規模・人数」の数値表を白紙再現できるまで反復(管理体制は最優先)。
2. 「巡視頻度」と「○か月/○年の保存期間」を語呂で固める(混同が失点源)。
3. 健診・面接指導・ストレスチェックの「頻度・対象・申出の要否・記録保存・報告(50人)」を一覧化。
4. 計画届の3つの期限(30日/30日/14日)と届出先(署長/厚労大臣)をセットで暗記。
5. 選択式対策として、目的条文(自主的活動・快適な職場環境)と各種定義の語句を赤字で押さえる。
6. 2章(労働災害防止計画)・8章(改善計画)は深追いせず、択一の総合問題対応として要点のみ確認。

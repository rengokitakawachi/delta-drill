# 📚 社労士 過去問分析 Vault

社労士試験合格のための、**過去問ベースの出題傾向分析**を科目ごとに蓄積するObsidian Vaultです。

## 使い方
- このフォルダ（`obsidian/`）を **Obsidianで「フォルダをVaultとして開く」** で開けます。
- スマホは **Obsidian Mobile** ＋ GitHub同期（Obsidian Git や Working Copy）で、いつでも閲覧できます。
- ○×問題集アプリと対応：分析で判明した**頻出論点**を、アプリ側の **🔥頻出バッジ・出題の重み付け** に反映しています。

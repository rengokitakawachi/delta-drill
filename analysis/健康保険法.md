# 健康保険法 過去問・選択式分析ノート

## サマリー

健康保険法は「保険給付」を軸に、その前提となる「適用・標準報酬」と、財源となる「費用負担・保険料」がバランスよく問われる科目。択一では保険給付（特に傷病手当金・高額療養費・出産関係）の支給要件と数値が繰り返し問われ、選択式では目的条文・標準報酬の上下限・各種給付の額（出産育児一時金50万円・埋葬料5万円）・期間（待期3日、通算1年6か月、20日、2年）が狙われる。数値・期間・割合を正確に暗記することが合否を分ける。

## 頻度ランキング（重要度の高い順）

1. 傷病手当金（支給額3分の2／待期3日／通算1年6か月）
2. 標準報酬月額（50等級・上限139万）／標準賞与額（年573万）
3. 保険給付の通則・併給調整
4. 高額療養費（自己負担限度額・多数回該当）
5. 出産に関する給付（出産育児一時金50万円・出産手当金）
6. 任意継続被保険者（2か月以上・20日以内・2年）
7. 保険料（一般保険料率30〜130／1000・労使折半・任意継続は全額）
8. 被保険者の資格得喪・適用拡大（短時間労働者）
9. 被扶養者（年収130万・180万・2分の1）
10. 死亡に関する給付（埋葬料5万円）
11. 資格喪失後の給付（継続給付1年以上・出産育児一時金6か月）
12. 保険者（協会700人／組合3000人・厚労大臣の業務）
13. 不服申立て・時効（いずれも2年）

## 重要度マップ

- 重要度3（最頻出）: 被保険者 / 任意継続被保険者 / 標準報酬月額 / 標準賞与額 / 療養の給付 / 傷病手当金 / 出産に関する給付 / 高額療養費 / 保険料
- 重要度2（頻出）: 総則 / 保険者(協会) / 保険者(組合) / 適用事業所 / 被扶養者 / 一部負担金 / 療養費 / 死亡に関する給付 / 資格喪失後の給付 / 日雇特例被保険者 / 費用負担 / 不服申立て / 時効 / 通則
- 重要度1（基礎・たまに出題）: 特例退職被保険者 / 入院時食事療養費 / 入院時生活療養費 / 保険外併用療養費 / 訪問看護療養費 / 移送費 / 高額介護合算療養費 / その他

## 🔥 頻出論点の押さえどころ

- **傷病手当金**: 額＝支給開始日以前直近12か月の標準報酬月額平均の30分の1の3分の2。待期は連続3日（有給・公休日も含む）、4日目から支給。支給期間は支給開始日から「通算して」1年6か月（令和4年改正、無支給期間は減算されない）。
- **標準報酬月額・標準賞与額**: 月額は50等級、下限58,000円・上限1,390,000円（139万）。標準賞与額は1,000円未満切捨て、年度（4/1〜翌3/31）累計573万円が上限。随時改定は固定的賃金変動で2等級以上の差。
- **出産に関する給付**: 出産育児一時金は原則48万8,000円、加算対象出産は1.2万円加算で50万円。出産手当金は産前42日（多胎98日）・産後56日、額は傷病手当金と同じ3分の2。
- **一部負担金割合**: 70歳未満3割、義務教育就学前2割（給付8割）、70歳以上は原則2割・現役並み所得者3割。
- **任意継続被保険者**: 資格喪失日の前日まで継続2か月以上、申出は20日以内、資格は2年で喪失、保険料は全額自己負担。
- **保険料・費用負担**: 協会けんぽ一般保険料率は1000分の30〜130（都道府県単位）。労使折半。国庫補助は給付費等の1000分の130〜200。
- **保険者の設立要件**: 健康保険組合は単独700人以上・共同3000人以上、被保険者の2分の1以上の同意＋厚労大臣の認可。
- **埋葬料**: 一律5万円（家族埋葬料も同額）。受給者がない場合は埋葬を行った者に埋葬費（額は埋葬料が上限）。
- **時効・不服申立て**: 保険料・保険給付を受ける権利の消滅時効は2年。資格・標準報酬に関する審査請求の除斥期間も2年。

## 近年傾向

- 令和4年改正の傷病手当金「通算1年6か月」が定着論点として問われやすい。
- 出産育児一時金の引上げ（50万円）など改正絡みの数値は要注意。
- 短時間労働者への適用拡大（特定適用事業所・2分の1以上同意対象者）が継続的に出題。
- 高額療養費・高額介護合算療養費は計算過程よりも仕組み・所得区分・多数回該当の知識問題が中心。
- 選択式は目的条文（国民の生活の安定と福祉の向上）と数値の組合せが定番。

## 学習アクション

1. 数値・期間・割合を一覧表で暗記（50等級/139万/573万/3割/2割/3分の2/待期3日/1年6か月/20日/2年/50万円/5万円/30〜130）。
2. 傷病手当金・出産手当金・高額療養費の支給要件と調整（併給調整）をセットで整理。
3. 任意継続・資格喪失後の継続給付の要件（2か月／1年／6か月）を混同しないよう区別。
4. 保険者の設立・解散要件（700人／3000人／2分の1／3分の2）と認可主体（厚労大臣）を確認。
5. 選択式対策として目的条文・基本的理念を一字一句で暗唱できるようにする。
